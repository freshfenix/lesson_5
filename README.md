# Lesson 3

## Опис проєкту

Це навчальний проєкт уроку 3, який демонструє створення багатосторінкового веб-сайту з використанням HTML та CSS. Сайт включає головну сторінку та кілька додаткових сторінок, таких як статті, каталог, контакти, галерея, навігація та реєстрація.

## Структура проєкту

- `index.html` - Головна сторінка сайту
- `pages/` - Папка з додатковими сторінками
  - `articles.html` - Сторінка статей
  - `catalog.html` - Сторінка каталогу
  - `contacts.html` - Сторінка контактів
  - `gallery.html` - Сторінка галереї
  - `page-nav.html` - Сторінка навігації
  - `registration.html` - Сторінка реєстрації
  - `articles/` - Підпапка з окремими статтями
    - `article1.html`
    - `article2.html`
    - `article3.html`
- `styles/` - Папка зі стилями
  - `reset.css` - Скидання стилів
  - `style.css` - Основні стилі сайту
- `fonts/` - Папка з шрифтами
- `images/` - Папка із зображеннями

## Як запустити

1. Клонуйте репозиторій або завантажте файли проєкту.
2. Відкрийте файл `index.html` у веб-браузері.

## Технології
